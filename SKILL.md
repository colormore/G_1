# Godot 游戏开发 Skills

本项目包含以下 6 个 Godot 游戏开发相关的 Skills，位于 `skills/` 目录下：

## 核心 Skills

| Skill | 路径 | 功能 |
|-------|------|------|
| godot-gdscript-patterns | `skills/godot-gdscript-patterns/` | Godot 4 生产级 GDScript 架构模式（状态机、对象池、组件系统等） |
| godot-best-practices | `skills/godot-best-practices/` | GDScript 编码最佳实践与官方标准对齐 |

## 扩展 Skills

| Skill | 路径 | 功能 |
|-------|------|------|
| godot-development | `skills/godot-development/` | Godot 场景搭建、MCP 工具链自动化 |
| godot-ui | `skills/godot-ui/` | Godot Control 节点系统、主题定制、游戏 UI 模式 |
| godot-optimization | `skills/godot-optimization/` | Godot 性能分析与瓶颈优化 |
| godot-asset-generator | `skills/godot-asset-generator/` | AI 自动化生成 2D 游戏素材并导入 Godot |
